# Dataset Notes

The default run uses:

- `earthquake_data/japan_2000_2023_query.csv`

This is the dataset meant to stay with the project, because it is the default input used by `main.py`.

The `earthquake_data/` folder also contains files from earlier exploration and supporting materials. They stay locally as reference data, but they are not loaded by `main.py` unless you point the script to them manually.
